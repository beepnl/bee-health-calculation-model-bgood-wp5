# Input data
## Measurement
- net_weight_kg (weight excluding beekeeper actions)

## Data interval
- 1 hour average

## Data period
- relative interval of 30 * 24 hours, looking back from now

## Data items
- data per hive

# Output data
- float between 0 and 1
